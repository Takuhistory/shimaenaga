# サイトマップ

## stay.example.com

宿泊者向けサイト。シマエナガニセコ、シマエナガ札幌の宿泊予約導線に集中する。

### 主要ページ

1. `/`
   - stay側トップ
   - ブランド導入
   - 施設一覧
   - Beds24空室検索ウィジェット設置エリア
   - シマエナガニセコ / シマエナガ札幌への導線

2. `/niseko/`
   - シマエナガニセコ施設ページ
   - 施設紹介
   - 写真ギャラリー
   - 設備・アメニティ
   - アクセス
   - 周辺観光
   - FAQ
   - Beds24予約ボタン
   - Beds24空室検索ウィジェット設置エリア

3. `/sapporo/`
   - シマエナガ札幌施設ページ
   - 施設紹介
   - 写真ギャラリー
   - 設備・アメニティ
   - アクセス
   - 周辺観光
   - FAQ
   - Beds24予約ボタン
   - Beds24空室検索ウィジェット設置エリア

4. `/faq/`
   - 宿泊者向けFAQ
   - 予約、チェックイン、設備、キャンセル、アクセス
   - Beds24側の予約・キャンセル案内と整合させる

5. `/access/`
   - ニセコ、札幌のアクセス情報まとめ
   - 空港、駅、車、公共交通

6. `/journal/`
   - 北海道観光
   - ニセコ・札幌周辺情報
   - 季節の過ごし方

7. `/privacy-policy/`
   - プライバシーポリシー

8. `/commercial-transaction/`
   - 特定商取引法に基づく表記
   - 予約・決済の実務はBeds24側と整合させる

### stay側の導線ルール

- 主CTAは「Beds24で予約」「空室を確認」に統一する
- 民泊運営相談のCTAは主導線にしない
- 施設選択から予約までを短くする
- 写真、設備、アクセス、FAQを予約ボタンの近くに置き、不安解消から予約へつなげる

## company.example.com / consulting.example.com

物件オーナー向けサイト。民泊運営代行、立ち上げ支援、Beds24導入・OTA運用支援の問い合わせ獲得に集中する。

### 主要ページ

1. `/`
   - company側トップ
   - 会社の信頼性
   - 支援メニュー
   - 実績・対応範囲
   - 相談CTA

2. `/management/`
   - 民泊運営代行
   - ゲスト対応
   - 清掃連携
   - 価格調整
   - レビュー改善
   - OTA運用

3. `/startup-support/`
   - 民泊立ち上げ支援
   - 物件確認
   - 設備・備品設計
   - 掲載準備
   - 運営体制づくり

4. `/beds24-ota/`
   - Beds24導入・OTA運用支援
   - Beds24初期設定
   - OTA連携
   - 料金・在庫運用
   - 公式サイト予約導線の設計

5. `/cases/`
   - 運営事例
   - 改善事例
   - 公開可能な範囲の実績
   - 写真、レビュー、稼働改善など

6. `/company/`
   - 会社概要
   - 代表・運営体制
   - 対応エリア
   - 事業方針

7. `/faq/`
   - オーナー向けFAQ
   - 費用、契約、対応範囲、導入までの流れ

8. `/contact/`
   - 問い合わせ
   - 初回相談
   - 物件情報、相談内容、希望支援範囲を入力できるフォーム

9. `/privacy-policy/`
   - プライバシーポリシー

### company側の導線ルール

- 主CTAは「民泊運営を相談する」「初回相談を申し込む」に統一する
- 宿泊予約CTA、Beds24予約ボタンは置かない
- Beds24は予約対象ではなく、導入・運用支援のサービス文脈で扱う
- 会社概要、実績、支援内容、相談の流れを近接させ、信頼形成から問い合わせへつなげる
